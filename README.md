# movie-py-cli
A simple script for fetching movie details right from the terminal

# Requirements
 1. sys 
 2. requests 
 3. bs4 
 4. time  
 5. colorama 
 6. urllib 
 7. os 
 8. demjson
<b> Note:- You can intall then using sudo pip install <name of the package></b>
