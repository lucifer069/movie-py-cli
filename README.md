# movie-py-cli
==============

A simple script for fetching movie details from IMDB.com using cli.

# Requirements
==============

 1. Requests
 2. BeautifulSoup4

# Packages Installation
=======================

Simply install the required packages with pip as below::

    $ pip install -r requirements.txt
